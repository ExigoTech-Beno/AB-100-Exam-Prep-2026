## How to Use Each Notebook File

1. **Open the file** to get the notebook title, audio focus prompt, and sources
2. **Create a new notebook** in NotebookLM with the given title
3. **Add all sources** listed — paste URLs directly (Microsoft Learn, YouTube, GitHub, blogs all work)
4. **Generate an Audio Overview** using the focus prompt in the Customize dialog
5. **Use the chat** for active study — quiz yourself, ask scenario questions, request comparisons
6. **Generate Flashcards or a Study Guide** from the Studio panel for additional practice
